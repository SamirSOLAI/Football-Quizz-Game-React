# Football Quiz (React)

A simple multiple-choice quiz app about Football, built with React.
The goal of this project was to practice React basics: components, state management, conditional rendering, and user interactions.

## Features
- Multiple-choice questions (4 options)
- Selected option highlight
- Submit to move to the next question
- Final score displayed at the end
- "See your results" button when the quiz is finished

## Tech Stack
- React
- JavaScript (ES6)
- CSS
